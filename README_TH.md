
# VNIX Order Management

- เปลี่ยนชื่อระบบจาก VNIX Order Orchestrator → **VNIX Order Management**
- นาฬิกาไทย พ.ศ. อัปเดตแบบเรียลไทม์
- ช่องวันที่เป็นปฏิทิน (input type=date)
- เพิ่มสถานะ "แพ็คแล้ว" และตัวกรอง
- ปุ่ม KPI กดเพื่อกรองได้
- ปุ่ม "ล้างตัวกรองตาราง" เคลียร์ตัวกรอง DataTables
- เพิ่มคอลัมน์ **AllQty** (ยอดรวมที่ลูกค้าสั่งของ SKU นั้นในทุกแพลตฟอร์ม)
- สถานะ SLA: วันนี้/พรุ่งนี้/เลยกำหนด (และอัปเดตทุก 1 นาที)
- อนุญาตให้ "กดรับ" แม้เป็นสถานะสินค้าน้อย (เตือนยืนยัน)
- ระบบล็อกอิน/ผู้ใช้ (เริ่มต้น: admin / admin123) + บันทึกผู้กดรับ
- รายงานคลัง: แสดงเฉพาะที่กดรับแล้ว, เลือกแพลตฟอร์ม/ร้าน/ขนส่ง, ช่องเบอร์โทรขนส่ง
